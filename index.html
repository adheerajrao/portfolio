Idea & behind the scenes

Designers and developers always tend to procrastinate projects we feel are of less importance and creating our own website tops the list. Although sometimes it is purely our workaholic tendencies that keep us busy on a project or two, we never feel the need to give that much time to our own website.

When I decided to design and develop my website I had to make sure that I give it ample of time and dedication as I didn’t wanted to end up being a part of "The shoemaker's kids have no shoes" phenomena.

Technical Stuff

This website is a responsive website hand-crafted from the ground up, built on the Foundation framework using HTML5, CSS3 & the jQuery library. I have used parallax.js for parallax effects, jQuery easing library for the cool transitions throughout the website. Viewport element reveal transitions are done using scrollReveal.js. Pictographic scalable vector icons are from the Font Awesome library. Animations and Interactive buttons were created using animate.css. Fonts used on the website are Raleway, Roboto, Open Sans, & Lobster served by Google Fonts. Last but not the least, entire project is coded with Brackets and hosted on Github.
